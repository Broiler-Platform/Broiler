# Broiler.UI Human Review

**Status:** PENDING

Broiler.UI Phase 0 was produced as planning and characterization material. It
is not a production-ready UI runtime and does not yet expose public control
contracts.

Before preview use, human review should cover:

- the ADR decisions in `docs/adr`;
- the ownership table and dependency graph in `docs/phase0/decisions.md`;
- the current-state inventory against the latest application and
  Broiler.Graphics checkouts;
- the browser chrome baseline artifacts; and
- the Phase 1 API scaffolding before it becomes a compatibility promise.
